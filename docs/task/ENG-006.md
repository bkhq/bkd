# ENG-006 ACP plan 与 diff 映射到现有产品 UI

- **status**: completed
- **priority**: P0
- **owner**: codex
- **plan**: PLAN-015
- **created**: 2026-03-13

## 描述

当前 ACP 已具备标准化的 tool-call action/result 配对，但仍缺少两块关键产品态映射：

1. ACP `plan` update 尚未映射到现有 `task-plan` UI。
2. ACP tool content 中的 `diff` 尚未映射到现有 patch / compare 渲染器。

## 调查结果

- ACP `plan` 目前在 `acp-client.ts` 中仅被归一化为 `system-message`，前端不会把它当任务计划卡片渲染。
- 前端当前只有 `TodoWrite` 才会触发 `task-plan` 重建，逻辑位于 `use-chat-messages.ts`。
- ACP `tool_call.content` 中的 `diff` 当前只被渲染成简短文本摘要，但原始结构已经可进入 `toolDetail.raw`。
- 前端已有可复用 diff 渲染能力：`CodeRenderers.tsx`、`DiffPanel.tsx` 以及 `ToolItems.tsx`。
