## Description

Add a project-level whiteboard feature that renders an interactive mindmap for project planning. Each project has one whiteboard. Nodes are stored individually in the database (one record per node) with `parentId` tree structure.

Key capabilities:
- Auto-layout mindmap with connecting lines (@xyflow/react + elkjs)
- Rich card nodes with inline markdown editing
- Node-level AI interaction via "Ask AI" popover (quick actions + auto-generated questions)
- AI interaction reuses existing engine system via a bound visible issue (follow-up turns)
- AI-recommended issue generation from node subtrees
- Full REST API for external BKD skill access
