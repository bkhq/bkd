# ENG-004 拆分 ACP agent 定义为独立文件

- **status**: completed
- **priority**: P1
- **owner**: codex
- **plan**: PLAN-013
- **created**: 2026-03-13

## 描述

当前 `apps/api/src/engines/executors/acp/agents.ts` 同时承担 ACP agent registry、命令解析、鉴权探测、模型发现与 model 前缀解析等职责。随着后续继续接入更多 ACP agent，单文件会快速膨胀，维护成本增加。

## 调查结果

- 当前 ACP 已支持 `gemini` 与 `codex` 两个 agent，agent 定义集中在一个文件中。
- `AcpExecutor`、测试和模型发现逻辑都通过 `./agents` 入口依赖这些定义。
- 将每个 agent 拆成独立文件后，可以在不改变 `acp` engine 外部 API 的前提下，把 agent 特定逻辑局部化。
