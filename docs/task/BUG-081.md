# BUG-081 修复客户端设置弹窗横向滚动

- status: completed
- priority: P2
- owner: codex
- createdAt: 2026-03-03 01:58 UTC
- updatedAt: 2026-03-03 02:00 UTC

## Summary
客户端设置弹窗出现横向滚动条，需要定位具体溢出元素并修复，使设置内容在常见桌面宽度和窄视窗下都不会产生多余横向滚动。

## Scope
- 定位 `AppSettingsDialog` 内触发横向溢出的布局或文本区域。
- 修改前端样式，消除不必要的横向滚动并保持现有交互。
- 进行一次前端构建或静态检查，确保改动无语法/类型回归。

## Acceptance Criteria
- 打开客户端设置弹窗时不再出现非预期横向滚动条。
- 长文本（如版本/构建信息）在窄宽度下不会撑开容器。
- 前端构建或测试通过（至少执行一项本地验证命令）。

## Notes
- 目标为最小改动，不引入额外 UI 重构。
