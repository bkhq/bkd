# ENG-011 ACP 前端恢复工具组展示

- **status**: completed
- **priority**: P1
- **owner**: codex
- **plan**: PLAN-020
- **created**: 2026-03-13

## 描述

当前 ACP 前端 renderer 已经独立出来，但工具调用按单条时间线项展示，产品阅读体验过碎。需要在保留 ACP 专用链路的前提下，恢复工具组作为主展示单元，同时继续让 `plan`、`assistant`、`system/session` 等非工具语义独立展示。

## 调查结果

- `use-acp-timeline.ts` 当前把每个 `tool-use` action/result 对直接映射为单个 `tool` timeline item，没有组装 `ToolGroupChatMessage`。
- `AcpTimeline.tsx` 当前逐个渲染 `AcpToolItem`，没有复用已有 `ToolGroupMessage` 分组 UI。
- 现有 `ToolGroupMessage` 已经具备折叠、分组摘要、组内工具卡片渲染等产品态能力，适合直接复用。

## 验证

- [x] ACP adapter 改为输出工具组
- [x] ACP renderer 改为复用 `ToolGroupMessage`
- [x] 定向 tests / lint 通过

## 实现结果

- `use-acp-timeline.ts` 现在会把连续 ACP 工具项聚合为单个 `tool-group`
- `AcpTimeline.tsx` 已改为直接复用现有 `ToolGroupMessage`
- `plan`、`assistant`、`system/session` 等非工具语义仍保持独立展示
