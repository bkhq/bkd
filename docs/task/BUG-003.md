# BUG-003 升级不要自动下载，只提示新版本

- **status**: completed
- **priority**: P1
- **owner**: (未分配)
- **createdAt**: 2026-03-08 19:35

## 描述

当前升级系统在启动时和每小时定期检查中，如果发现新版本会自动下载到 `data/updates/`。用户希望改为只检查并提示有新版本，下载需要用户手动点击触发。

**影响位置：**

- `apps/api/src/upgrade/service.ts` — `checkAndAutoDownload()` 函数在启动和定时器中自动下载
- `apps/frontend/src/i18n/en.json` / `zh.json` — 提示文案需要更新

## 进行时描述

移除升级自动下载，改为仅提示
