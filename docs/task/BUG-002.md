# BUG-002 ChatInput 刷新按钮不可用

- **status**: completed
- **priority**: P1
- **owner**: (未分配)
- **createdAt**: 2026-03-08 19:30

## 描述

ChatInput 的刷新按钮（RefreshCw 图标）点击后清空日志但不会重新拉取。原因是 `useIssueStream` hook 中的日志拉取 effect 的依赖数组遗漏了 `_refreshCounter`。

`refreshLogs()` 调用时：

1. `clearLogs()` 清空所有日志状态 ✅
2. `setRefreshCounter(c => c + 1)` 递增计数器 ✅
3. 日志拉取 effect 不会重新执行，因为 `_refreshCounter` 不在依赖数组中 ❌

**影响位置：**

- `apps/frontend/src/hooks/use-issue-stream.ts:383` — 依赖数组缺少 `_refreshCounter`
